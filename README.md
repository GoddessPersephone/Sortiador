# Gerador de Número Aleatório

Um projeto web simples que gera um número aleatório entre 1 e 100 ao clicar em um botão. Feito com HTML, CSS e JavaScript e configurado para rodar em um servidor ASP.NET Core minimalista.

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
- ASP.NET Core para hospedagem estática

## Como Executar

### Pré-requisitos
- Visual Studio 2022 com .NET 6 ou superior instalado
